### Keyword

+ 序列推导：List Comprehension

+ 生成器表达式：Generator Expression

### 序列

1. 容器序列：存放的任意类型的对象的引用
   + list
   + tuple
   + collections.dqueue

2. 扁平序列：存放的是值
   + str
   + bytes：字节
   + bytearray
   + array.array
   + memoryview

3. 可变序列
   + list
   + collections.dqueue
   + bytearray
   + array.array
   + memoryview

4. 不可变序列
   + tuple
   + str
   + bytes

### 可迭代对象

**可迭代对象（iterable）**就是“可以被 for 循环逐个取值”的对象

常见有：list、tuple、str、dict、set、range、文件对象等

### 推导comprehension

+ 列表推导List Comprehension:listcomps
+ 生成器比倒是Generator Expression：genexps
