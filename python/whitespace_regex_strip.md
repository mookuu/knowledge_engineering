### 正则 `\s` 与 `strip()` 空白字符的区别

两者都处理「空白」，但**定义来源、用途、范围**不同，不能混为一谈。

---

#### 定义来源

| | 正则 `\s` | `str.strip()` / `lstrip()` / `rstrip()` |
|--|-----------|----------------------------------------|
| 依据 | `re` 模块的 Unicode 空白类 | `str.isspace()` |
| 典型 API | `re.search(r"\s+", s)` | `"  hello\n".strip()` |

---

#### 常见都会匹配的字符

| 字符 | 写法 | `\s` | `strip()` 默认 |
|------|------|:----:|:--------------:|
| 空格 | ` ` | ✅ | ✅ |
| 制表符 | `\t` | ✅ | ✅ |
| 换行 | `\n` | ✅ | ✅ |
| 回车 | `\r` | ✅ | ✅ |
| 换页 | `\f` | ✅ | ✅ |
| 垂直制表 | `\v` | ✅ | ✅ |
| 不间断空格 | `\u00a0` (NBSP) | ✅ | ✅ |
| 全角空格 | `\u3000` | ✅ | ✅ |

日常 ASCII / 常见 Unicode 空白，两者行为**基本一致**。

---

#### 主要区别

| 维度 | `\s` | `strip()` 默认 |
|------|------|----------------|
| **用途** | 在模式中**匹配**空白 | 从字符串**两端删除**空白 |
| **作用位置** | 任意位置（由模式决定） | **仅两端**，中间不动 |
| **是否同一集合** | 与 `isspace()` **非常接近**，但不保证 100% 相同 | 以 `isspace()` 为准 |
| **自定义** | 用字符类 `[ \t\n\r]` 等收窄 | `strip(chars)` 传入**任意字符集**（不限于空白） |

**零宽字符**（如 `\u200b`）通常：

- `isspace()` → `False` → `strip()` **不删**
- `\s` 一般也**不匹配**

---

#### 行为示例

```python
import re

# 匹配：中间空白也会被 \s+ 命中
re.findall(r"\s+", "a b\tc\nd")   # [' ', '\t', '\n']

# 删除：只剥两端
"a b c".strip()                   # 'a b c'

# Unicode 空白
"\u00a0hello\u3000".strip()       # 'hello'
re.findall(r"\s", "\u00a0a\u3000") # ['\xa0', '\u3000']

# 只要 ASCII 空白：显式写死，别依赖默认语义
s.strip(" \t\n\r\f\v")
re.sub(r"[ \t\n\r\f\v]+", " ", s)
```

```python
# strip(chars) 不是「空白」，而是「要删的字符集合」
"...hello...".strip(".")          # 'hello'
```

---

#### 选型建议

| 需求 | 推荐 |
|------|------|
| 正则里匹配任意空白 | `\s` 或 `\s+` |
| 只要空格 / Tab / 换行 | `[ \t\n\r]`（更可控） |
| 去掉字符串首尾空白 | `strip()` / `lstrip()` / `rstrip()` |
| 整段必须符合格式 | `fullmatch` + 明确字符类 |
| 非空白 | 正则 `\S` |

---

#### 相关 API（对照）

| API | 说明 |
|-----|------|
| `re.match` | 仅从字符串**开头**匹配（隐式 `^`） |
| `re.search` | 在**任意位置**找第一个匹配 |
| `re.fullmatch` | **整串**必须完全匹配（`^...$`） |
| `re.split(pat, s)` | 按模式切分；末尾被分隔符切断时可能产生 `''` |
| `strip()` | 默认 `isspace()` 字符，从两端删除 |

---

#### 记忆

- **`\s`**：正则元字符，表示「空白类」→ **匹配**
- **`strip()`**：字符串方法，默认 `isspace()` → **删除两端**
- 日常常见空白两者差不多；要严格只处理 ASCII 空白时，**两边都显式写字符集**
