# Vi / Vim 常用命令备忘

> **适用**：终端文本编辑，Linux 服务器、WSL、Git commit 等场景。  
> **平台**：跨平台（Linux / macOS / WSL）。  
> **官方**：[Vim 文档](https://www.vim.org/docs.php)

---

## 一、模式切换

| 操作 | 按键 | 说明 |
|------|------|------|
| 进入插入模式 | `i` | 在光标前插入 |
| 进入插入模式（行首） | `I` | 跳到行首并插入 |
| 进入插入模式（行尾） | `A` | 跳到行尾并插入 |
| 进入插入模式（下一行） | `o` | 当前行下方新开一行 |
| 进入插入模式（上一行） | `O` | 当前行上方新开一行 |
| 返回普通模式 | `Esc` | 最常用的退出方式 |
| 返回普通模式 | `Ctrl + [` | 替代 Esc |
| 可视模式 | `v` | 字符选择 |
| 可视行模式 | `V` | 行选择 |
| 可视块模式 | `Ctrl + v` | 矩形块选择 |
| 命令模式 | `:` | 输入 Ex 命令 |

## 二、基本移动

| 按键 | 说明 |
|------|------|
| `h` `j` `k` `l` | ← ↓ ↑ → |
| `w` | 下一个单词开头 |
| `b` | 上一个单词开头 |
| `e` | 单词结尾 |
| `0` | 行首 |
| `^` | 行首（非空字符） |
| `$` | 行尾 |
| `gg` | 文件开头 |
| `G` | 文件末尾 |
| `nG` | 跳转到第 n 行（如 `42G` → 第 42 行） |
| `Ctrl + f` | 向下翻一页 |
| `Ctrl + b` | 向上翻一页 |
| `%` | 匹配括号 `()` `{}` `[]` 间跳转 |

## 三、编辑操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 删除光标处字符 | `x` | 相当于 Delete |
| 删除光标前字符 | `X` | 相当于 Backspace |
| 删除（剪切）选中 | `d` + 移动命令 | 如 `dw` 删一个词，`dd` 删一行 |
| 删除并进入插入 | `c` + 移动命令 | 如 `cw` 改一个词，`cc` 改一行 |
| 复制（yank） | `y` + 移动命令 | 如 `yw` 复制一词，`yy` 复制一行 |
| 粘贴 | `p` | 光标后粘贴 |
| 粘贴（光标前） | `P` | 光标前粘贴 |
| 撤销 | `u` | 撤销上一步 |
| 重做 | `Ctrl + r` | 恢复撤销 |
| 缩进 | `>>` / `<<` | 向右/左缩进 |
| 自动缩进 | `==` | 对齐当前行 |
| 大小写转换 | `~` | 切换光标处字符大小写 |
| 替换一个字符 | `r` | 替换当前字符并返回普通模式 |
| 连续替换 | `R` | 进入替换模式 |

## 四、搜索与替换

| 操作 | 按键/命令 | 说明 |
|------|----------|------|
| 向前搜索 | `/pattern` | 如 `/nginx` 搜索 nginx |
| 向后搜索 | `?pattern` | 反向搜索 |
| 下一匹配 | `n` | 跳到下一个 |
| 上一匹配 | `N` | 跳到上一个 |
| 高亮取消 | `:noh` | 取消搜索高亮 |
| 替换 | `:s/old/new/g` | 替换当前行所有匹配 |
| 全局替换 | `:%s/old/new/g` | 替换全文所有匹配 |
| 全局替换（确认） | `:%s/old/new/gc` | 逐个确认替换 |

## 五、多文件操作

| 命令 | 说明 |
|------|------|
| `:e file` | 打开文件 |
| `:w` | 保存 |
| `:w !sudo tee %` | 以 sudo 权限保存（忘了 sudo 启动时） |
| `:q` | 退出 |
| `:q!` | 强制退出（不保存） |
| `:wq` 或 `ZZ` | 保存并退出 |
| `:x` | 保存并退出（仅修改时写入） |
| `:bn` | 下一个缓冲区 |
| `:bp` | 上一个缓冲区 |
| `:bd` | 关闭当前缓冲区 |
| `:ls` | 列出所有缓冲区 |
| `:sp file` | 水平分屏打开 |
| `:vsp file` | 垂直分屏打开 |
| `Ctrl + w w` | 切换分屏窗口 |
| `Ctrl + w q` | 关闭当前分屏 |

## 六、常用技巧

### 6.1 重复上次操作

`.` — 重复上一次修改命令，非常实用。

### 6.2 使用数字重复

几乎所有命令前可加数字：`5dd` 删 5 行，`3yy` 复制 3 行，`10j` 向下 10 行。

### 6.3 结合系统剪贴板

```bash
"+y    # 复制到系统剪贴板
"+p    # 从系统剪贴板粘贴
```

需 Vim 支持 `+clipboard`，可用 `vim --version \| grep clipboard` 检查。

### 6.4 可视模式下操作

1. `v` 选中文本
2. 按 `y` 复制 / `d` 剪切 / `~` 大小写转换
3. 选中后按 `:` 可对选中范围执行命令（自动插入 `:'<,'>`）

### 6.5 Git commit 编辑

```bash
# 默认进入插入模式即可写 commit message
# 写完后：
Esc → :wq  （或 ZZ）
# 放弃提交：
Esc → :q!
```
